# 儿童识字小报生成器

一个基于Nano Banana Pro AI模型的网页应用，专为5-9岁儿童制作有趣的识字学习材料。

## 功能特点

- 🎨 **智能生成**：根据选定主题自动生成包含15-20个词汇的识字小报
- 📚 **丰富主题**：包含超市、医院、公园、学校、动物园、交通等常见场景
- 🔤 **拼音标注**：每个汉字都配有拼音，方便儿童学习
- 🖼️ **高清输出**：生成高质量的PNG格式图片，适合打印和分享
- 💾 **本地存储**：API密钥安全地保存在浏览器本地

## 使用方法

### 1. 获取API密钥

访问 [KIE AI API Key管理页面](https://kie.ai/api-key) 注册并获取您的API密钥。

### 2. 使用生成器

1. 在浏览器中打开 `index.html`
2. 输入您的API密钥（会自动保存）
3. 选择一个主题（如：超市、医院等）
4. 输入小报标题（如：《走进超市》）
5. 点击"生成识字小报"按钮
6. 等待1-3分钟生成完成
7. 下载并保存图片

## 技术实现

### 提示词生成
- 基于预定义的词汇库，每个主题包含核心角色、常见物品和环境装饰三类词汇
- 自动生成符合儿童认知特点的详细绘图提示词
- 包含画面构图、色彩、风格等具体要求

### API集成
- 使用Nano Banana Pro模型的图像生成API
- 支持异步任务创建和状态轮询
- 完善的错误处理和用户反馈

### 用户体验
- 响应式设计，适配各种设备
- 实时进度显示
- 词汇预览功能
- 一键下载生成的图片

## 成本控制

为控制使用成本，系统默认使用以下参数：
- **分辨率**：1K（平衡质量与成本）
- **宽高比**：3:4（适合A4竖版打印）
- **格式**：PNG（支持透明背景）

## 注意事项

3. **网络要求**：需要稳定的网络连接访问API服务
4. **浏览器兼容**：建议使用Chrome、Firefox等现代浏览器

## 常见问题

**Q: 为什么生成失败？**
A: 请检查API密钥是否正确，账户是否有足够余额，网络连接是否正常。

**Q: 可以修改词汇列表吗？**
A: 目前版本使用预定义词汇库，您可以通过编辑 `data/vocabulary.json` 文件来自定义词汇。

**Q: 生成的图片可以商用吗？**
A: 请参考Nano Banana Pro的使用条款，遵守相关的版权规定。
